# WordPress Plugin - wp_excerpt_rss

## wp_excerpt_rss で出来ること

RSSの配信を抜粋配信にし、画像を1つだけ表示します。

## 対応Version

基本的にWordPress 3.x系で確認していますが、もしかすると2.x系でも動くやも知れません。
